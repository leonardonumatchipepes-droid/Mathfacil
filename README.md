# Mathfacil
RESUMO A presente investigação apresenta o Math Fácil, um aplicativo educacional desenvolvido como ferramenta digital auxiliar no processo de ensino e aprendizagem da Matemática nos diferentes subsistemas de ensino em Angola, nomeadamente o Ensino Primário, o I Ciclo do Ensino Secundário, o II Ciclo do Ensino Secundário e o Ensino Superior.
